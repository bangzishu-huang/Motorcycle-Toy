# Motorcycle-Toy
A complex assembly of 3d parts that when fully assembled becomes a motorcycle. The final product is able to move and turn. 

I made this project to push the boundaries of my skills in AutoDesk Inventor. Although I didn't expect that the project would be pretty difficult as I was working on uneven surfaces (so I just liked 50 work planes). Overall, I think I was able to more effeciently navigate through Inventor and become more fluid. I really enjoyed the process, even when my brain was fried half the time.
